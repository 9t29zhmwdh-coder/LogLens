# Privacy Policy : LogLens

## Summary

LogLens processes all data locally. No data ever leaves your device.

## What I Collect

**Nothing.** LogLens does not collect, store, transmit, or share any user data.

## Data Processing

- All processing is RAM-only unless explicitly saved by the user
- No data is transmitted to external services
- All AI analysis runs locally via Ollama at localhost:11434
- No telemetry, no analytics, no crash reporting

## Storage

- User settings are stored in the OS application data directory
- No personal information in settings (only configuration values)
- No cloud sync

## Data Retention

Zero persistent data beyond user settings. LogLens retains no session data.
